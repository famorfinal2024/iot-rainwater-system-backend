# IoT-Based Rainwater Harvesting and Fertigation System

## Overview
This project is a **Django REST Framework (DRF) backend** for an IoT-powered rainwater harvesting and fertigation system for cornfields.  
It allows monitoring system data, managing schedules, and sending alerts to users.

## Features
- CRUD operations for SystemInfo, Schedule, and Alerts
- Token-based authentication for secure access
- RESTful API endpoints ready for mobile and web integration
- Admin panel for managing users and tokens

## Technologies Used
- Python 3.14
- Django 4.x
- Django REST Framework
- SQLite (default, can be changed)
- httpie for API testing
